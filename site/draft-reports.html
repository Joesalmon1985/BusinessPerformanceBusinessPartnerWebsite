Draft reports

Six worked examples of agent-assisted analytical briefs using public aggregate NHS data for Dorset HealthCare (RDY). Each is a first draft for human review — not official Trust reporting.

Demonstration caveat: Public aggregate data only. Not official Dorset HealthCare reports. Human review and local owner confirmation required before any operational use.

Public-data agent workflow examples

Each brief follows the same spine: question → data → key findings → agent summary → human checks → verify. Regenerate with Rscript site/R/03_render_public_reports.R. Source catalogue: DATA_SOURCE_REGISTER.csv.

How these briefs were created

  • Public NHS sources downloaded and catalogued in DATA_SOURCE_REGISTER.csv.
  • R scripts filter RDY rows and create public-data/processed/demo_*.csv extracts.
  • R/03_render_public_reports.R renders each brief: question, data used, Key findings explained, agent summary, human checks, verify appendix.
  • Every brief ends with explicit caveats and a human review gate.
