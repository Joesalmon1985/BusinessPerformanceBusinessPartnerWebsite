Worked example: AI-assisted Talking Therapies public-data briefing

IAPT monthly activity — RDY (April 2026)

Agent-assisted analytical brief — public-data demonstration: This is not an official Dorset HealthCare report. It uses public aggregate data only and requires human review and local owner confirmation before operational use.

What this report demonstrates

Summarising access, activity and waiting-style public IAPT measures with plain-English commentary, suppression handling, and descriptive monthly trends from the downloaded time series — without overclaiming recovery or clinical quality.

The question given to the agent

From public NHS Talking Therapies data for RDY, explain referrals, access and waiting measures in plain English, show trends where the time-series extract supports them, and list what outcome analysis would require.

Prompt excerpt

Filter IAPT monthly CSV to ORG_CODE2=RDY, Provider group.
Commentary on M001, M031, M019–M022, M053 and suppression.
Trend from downloaded time-series file only. No recovery inference. No causal claims.

Data used

  • NHS Talking Therapies Monthly Statistics (demo_talking_therapies.csv)
  • Time series (rdy_talking_therapies_talking_therapies_performance-april-2026_time_series.csv — 13 monthly periods)

2026-04-01 to 2026-04-30

RDY filter: ODS code RDY and case-insensitive trust name variants (Dorset HealthCare / Dorset Healthcare University NHS Foundation Trust). See site/public-data/metadata/filter_note_*.txt for per-source notes.

Agent process demonstrated

  1. Open demo_talking_therapies.csv and Provider RDY time-series extract
  2. Select M001, M031, M053 and waiting-band measures; count suppression
  3. Build Key figures explained with previous-period comparators — no invented access thresholds
  4. Render descriptive trend charts for referrals, access and six-week % where ≥2 months exist
  5. Filter to RDY using ODS code and Dorset HealthCare trust name variants (see filter notes)
  6. Detect current vs historic periods from reporting columns or stacked trend files
  7. Select key figures from named measures, top-N tables or source-presence checks — not exhaustive lists
  8. Decide trend analysis only when ≥2 comparable periods exist and values are not suppressed
  9. Avoid overclaiming — no causal language, no fabricated targets, no recalculated peer comparators
  10. Document verification paths and what a human reviewer must still confirm

How to read this report

  • Each row is one NHS Talking Therapies measure for RDY as provider (ORG_CODE2=RDY, GROUP_TYPE=Provider).
  • MEASURE_VALUE_SUPPRESSED is the published value for 2026-04-01 to 2026-04-30; '*' means suppressed.
  • M001 counts referrals received; M031 counts people accessing; M019–M022 count open referrals with no activity by waiting band.
  • Percentage measures (e.g. M053) require denominator understanding — do not treat as simple performance scores.
  • Monthly IAPT data are provisional — confirm refresh status. This brief does not infer clinical quality from access counts alone.
  • Worked example only — not official Dorset HealthCare reporting.

First-draft analysis

Public data shows 1,815 referrals received (M001) for 2026-04-01 to 2026-04-30. The time series (13 months) latest month is 1,815 vs 2,060 prior month — descriptive only.

Open referral 'no activity' bands (60–120 days selected) total 5,870 — the agent flags these for local pathway validation, not as a standalone waiting performance conclusion.

36 suppressed measures. Recovery/outcome analysis (M192, M186, etc.) requires separate definition checks — not inferred in this access brief.

Agent-generated observations

  • 1,815 referrals received (M001) — local validation required.
  • M001 trend available across 13 months in time-series extract.
  • Open referral no-activity bands (60–120d) sum to 5,870 — pathway context needed.
  • 36 suppressed measures — do not infer from missing cells.
  • Provisional monthly IAPT data — outcome measures exist in CSV but are not headline-charted here.

What cannot be concluded from this data

  • Recovery rates or clinical effectiveness from this access-focused brief
  • Enter-treatment waiting performance without local pathway definitions
  • Employment advisor subset performance unless explicitly in extract
  • Clinical quality from access or waiting counts without outcome definition checks
  • Causal explanation of referral or access trends from public monthly data alone

Questions for a Business & Performance Partner

  • Is this the same definition used locally?

    Public datasets may use national definitions. Local dashboards may use different filters, denominators or reporting dates.

  • Is this still the latest position?

    Public data may be a delayed publication. Local operational data may have moved on since the reporting period shown.

  • Who owns the narrative?

    The accountable service, finance, workforce, quality or BI owner should confirm the explanation before the finding is used.

  • Is this a performance issue, a data quality issue, or a service model issue?

    A public figure can be affected by coding, exclusions, small numbers, local pathways or publication rules.

  • Do local IAPT reporting figures match M001 for the same period?

    Public and local dashboards may use different refresh dates or referral definitions.

  • How are enter-treatment waits tracked locally vs public waiting bands?

    M019–M022 'no activity' bands may not match local RTT or enter-treatment definitions.

  • What did the monthly data quality report flag for this publication?

    IAPT submissions have known DQ checks — confirm before using figures in review.

How to verify these figures

  1. Open the linked demo CSV or audit file for the figure you are checking.
  2. Locate the RDY row using the filter notes and ODS code RDY.
  3. Confirm the value matches the published source — this demo does not recalculate NHS England comparators.
  4. Check metric definitions, provisional/final status and suppression before any operational use.

Figures trace to the demo Talking Therapies CSV, RDY-filtered processed extract and time-series file where cited. No derived ranks or peer medians are computed in this brief.
