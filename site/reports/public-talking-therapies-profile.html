NHS Talking Therapies access and waits brief — RDY provider rows

Public-data demonstration report: This is not an official Dorset HealthCare report. It uses public aggregate data only and requires human review and local owner confirmation before operational use.

What the agent was asked to do

Business question: From public NHS Talking Therapies data for RDY, explain referrals, access and waiting measures, show trends where the time-series extract supports them, and list what outcome analysis would require.

Dataset: NHS Talking Therapies Monthly Statistics (RDY provider rows)

This is a first-draft analytical brief for human review — not an approved performance report.

Scope: Provider / RDY rows only — not Dorset population or ICB-resident views.

Data used

  • NHS Talking Therapies (demo_talking_therapies.csv)
  • trend_talking_therapies_rdy.csv — six-month access stack
  • Time series (trend_talking_therapies_rdy.csv (or time-series fallback))
  • Period covered: Six available monthly data points: Aug 2025 – Apr 2026 (non-consecutive months in extract)
  • RDY filter: ODS code RDY; Dorset HealthCare trust name variants (see filter notes)
  • Historic trend data: Yes — 6 data points from trend_talking_therapies_rdy.csv (or time-series fallback)

What this report can and cannot tell us

This report can:

  • Describe referrals, access and waiting measures for RDY as provider.
  • Show descriptive month-on-month trends where the Provider time series supports them.

This report cannot:

  • Infer recovery or clinical outcomes from access measures alone.
  • Draw conclusions from suppressed cells or missing values.

For that reason, this should be treated as a first-draft prompt for review, not a final performance judgement.

Headline reading

  • Six-week access (M053) remains above the 75% national standard at 88%, but the selected trend window shows a falling pattern — review before it becomes a breach risk.
  • Self-referrals (M002: 1,370) account for ~75% of referrals (M001) — confirm whether this route mix is expected locally.
  • Open referrals with no activity: M019–M021 total 5,870; M019–M022 total 6,780 (including 910 over 120 days).
  • Six available monthly data points: Aug 2025 – Apr 2026 (non-consecutive months in extract)
  • This brief covers access and waits only — recovery and outcome measures need a separate analysis.

Priority review flags

  • M053 six-week access falling in selected trend window (latest 88%) — still above 75% standard but needs pathway-owner review.
  • 910 open referrals with no activity over 120 days (M022) — key waiting-band flag.

Why this is useful

  • The agent triangulated referrals, access standards, waiting bands and self-referral mix from public Provider/RDY rows.
  • It applied national access standards where documented and flagged where human pathway owners must confirm definitions.

Human validation checklist

  • Is this the same definition used locally? — Public datasets may use national definitions. Local dashboards may use different filters, denominators or reporting dates.
  • Is this still the latest position? — Public data may be delayed. Local operational data may have moved on since the reporting period shown.
  • Is the figure affected by data quality, suppression, coding or service model? — A public figure can be affected by small numbers, exclusions, coding rules or how services are configured.
  • Who owns the local narrative? — The accountable service, finance, workforce, quality or BI owner should confirm the explanation before use.
  • Is this suitable for board/service reporting, or only a prompt for further review? — These briefs are first drafts — confirm whether the figure is ready for formal reporting or needs more local work.

Bottom line

RDY remains above national six-week and eighteen-week access standards in the latest month, but six-week access is falling in the selected trend window and should be reviewed. Self-referrals dominate referral volume and waiting-band totals need pathway-owner validation. This is a useful first-draft access brief — not a complete Talking Therapies performance picture without outcomes.
